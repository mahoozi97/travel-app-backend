## Overview
This is the **backend of a Travel Ads platform** built using **Node.js, Express, and MongoDB**.  
It handles **travel ads, users**, providing a clean, secure, and well-documented REST API.

---

## Features

- RESTful API endpoints for managing:
  - **Users** (login, roles)
  - **Travel Ads** (create, read, update, delete)
- **Authentication & Authorization** with JWT
- **Input validation** and **error handling**
- **Aggregation queries** for filtering and sorting travel ads
- **Clean and maintainable code** following backend best practices
- **API Documentation** available via Postman

---

## Technologies Used

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB, Mongoose  
- **Authentication:** JWT (JSON Web Tokens)  
- **Documentation:** Postman Collection
